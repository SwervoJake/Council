# Sage — Recovery Framework

The structured approach to cognitive and physical recovery after high-load sessions.

---

## Recovery Types

### Type 1: Micro-Recovery (during session)
**When:** Session is ongoing but a decision pause is warranted
**Duration:** 5–15 minutes
**Protocol:**
1. Stop the active decision thread. Don't push.
2. Move physically — stand, walk, change location.
3. Box breathing: 4s in / 4s hold / 4s out / 4s hold. 5 rounds.
4. Drink water.
5. Return only when clarity feels restored.

**Sage flag:** `[SAGE: CHECK-IN]`

---

### Type 2: Session Recovery (between sessions)
**When:** Session has ended; preparing for the next
**Duration:** Minimum 2 hours, ideally overnight
**Protocol:**
1. Physical movement within 30 minutes of session end (20+ min walk is ideal)
2. No more complex decisions for the rest of the day
3. Good meal — protein and complex carbohydrates
4. Normal sleep — don't sacrifice sleep for "one more thing"
5. Review tomorrow's agenda before sleeping to allow unconscious processing

**Sage flag:** `[SAGE: REST]`

---

### Type 3: Deep Recovery (after extended high-load periods)
**When:** 5+ consecutive days of high-intensity decision-making
**Duration:** 24–48 hours of cognitive rest
**Protocol:**
1. No council sessions for 24 hours minimum
2. Low-stakes or administrative work only
3. Physical activity daily (not just sedentary rest)
4. Prioritize sleep above all else — catch up on sleep debt
5. Social connection if possible — conversation about non-work topics
6. Schedule the next high-stakes session at least 48 hours out

**Sage flag:** `[SAGE: LOAD]`

---

### Type 4: Decision Deferral (for specific decisions)
**When:** `[SAGE: DEFER]` is triggered for a specific decision
**Protocol:**
1. Write down: the decision, the options, and the current thinking
2. Close the session and the document
3. Sleep — at minimum one full night
4. Return with fresh context — reread the written summary, not from memory
5. Make the decision with fresh eyes
6. Research shows: >80% of decisions improve when re-evaluated after sleep

**Sage flag:** `[SAGE: DEFER]`

---

## Escalation to Professional Support
**When `[ESCALATE: PROFESSIONAL]` is triggered:**

Sage recognizes these signals go beyond wellbeing advisory:
- Persistent sleep disturbance (>2 weeks)
- Anxiety or panic that affects daily function
- Mood changes lasting >2 weeks
- Physical symptoms without clear lifestyle cause
- Burnout that doesn't respond to recovery protocols above

**Action:** Sage does not attempt to treat clinical-level concerns. The recommendation is to consult a licensed professional — doctor, therapist, or psychiatrist. Sage provides the flag; the operator decides the action.

## Minimum Effective Doses (reference)

| Recovery Need | MED | Alternative if time-constrained |
|--------------|-----|--------------------------------|
| Mental clarity | 20-min walk, no screens | 10-min walk + 5-min breathing |
| Sleep deficit | 60–90 min extension | 20-min nap (set alarm) |
| Acute stress | 5-min box breathing | 1-min cold water on face + 3 deep breaths |
| Cognitive fatigue | Overnight rest | 90-min away from screens |
| Decision quality | Sleep on it | 2-hour pause before deciding |
