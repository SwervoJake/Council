## Pattern: Diverge Before Converging
- **Confidence:** 0.5
- **Applications:** 0
- **Last updated:** 2026-03-04
- **Pattern:** Generate minimum 7 ideas without filtering before any curation begins. The best idea is rarely the first one. Premature curation kills the adjacent possibilities that live in ideas 5–9.
- **Conditions:** Any creative ideation session.
- **Counter-evidence:** Time-critical emergency responses where single best answer is required immediately.
- **Source:** Factory Council architecture specification

## Pattern: Always Include a Wildcard
- **Confidence:** 0.5
- **Applications:** 0
- **Last updated:** 2026-03-04
- **Pattern:** Every idea set ends with one option that deliberately breaks the brief's assumptions. The wildcard occasionally becomes the right answer. More often, it shifts the frame for evaluating the other options. Never omit it.
- **Conditions:** Every creative ideation session.
- **Counter-evidence:** Not applicable — the wildcard is a universal protocol.
- **Source:** Factory Council architecture specification

## Pattern: Reframe Wicked Problems
- **Confidence:** 0.5
- **Applications:** 0
- **Last updated:** 2026-03-04
- **Pattern:** For wicked problems (ambiguous framing, no clear answer), Muse questions the question before the council answers it. "Are we solving the right problem?" is often the most valuable creative intervention.
- **Conditions:** Tasks classified as WICKED complexity — ambiguous framing, no clear answer.
- **Counter-evidence:** Well-defined problems with clear success criteria — reframing wastes time.
- **Source:** Factory Council architecture specification
